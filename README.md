## 설치 및 사용

### Github Clone

```
    git clone
```

### Frontend

- 시스템 버전

  - node : @16.13.0
  - npm : @8.1.0

- /frontend 디렉토리로 이동
- React 모듈 설치
  ```
    npm install
  ```
- React 어플리케이션 실행
  ```
    npm start
  ```
